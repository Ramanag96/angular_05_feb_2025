Web Development
===============
-Developing a website
1. FrontEnd : User Interface (Look & Feel)
    HTML:  Content  (paragraph , dropdown , list , table , image, form)
    CSS:   Styling  (border,color,width,height,placement)
    Bootstrap: CSS framework (Predefined css classes)
    Javascript: Behaviour 
    Angular/React/Vuejs/NextJs : Javascript Frameworks
2. BackEnd : Develop REST APIs (Java,.net,python,PHP)
3. DataBase : Holds the data (Oracle , postgre,mysql , mongodb)
4. Server : where the app can be deployed (AWS,weblogic)
